# python-snake-game

Classic Snake game built with Python and Pygame, featuring score system, music, and cross-platform support.

## Features

- Snake movement with keyboard controls
- Collision detection (self collision)
- Score system with persistent record
- Multiple apple colors
- Background music system
- Wrap-around map
- Restart system after game over

## Technologies

- Python 3
- Pygame

## Installation

```bash
git clone https://github.com/giulio-caiazza/python-snake-game.git
cd python-snake-game
pip install -r requirements.txt
```

## Run

```bash
python src/SnakeGame.py
```
