# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-16

### Added

* Snake movement system
* Apple spawning system
* Score tracking
* Persistent high score system
* Background music support
* Game Over screen
* Restart functionality
* Random snake colors
* Random apple colors

### Technical

* Python implementation using Pygame
* Cross-platform score storage
* Object-oriented architecture
* Thread-based update system
